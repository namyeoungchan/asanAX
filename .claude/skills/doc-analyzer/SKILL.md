# doc-analyzer — 문서 구조 분석기

외부 패키지 설치 없이(파이썬 표준 라이브러리만) 문서의 구조와 통계를 뽑아냅니다.
긴 문서를 읽기 전에 "무엇이 어떻게 들어 있는지" 먼저 파악하는 용도예요.

## 언제 쓰나
- 처음 보는 문서의 분량·구조를 빠르게 파악할 때
- 마크다운 문서의 목차(헤딩 트리)를 추출할 때
- CSV/JSON의 컬럼·키 구조(스키마)를 확인할 때
- Word(.docx) 문서의 문단 수·표 개수를 셀 때

## 사용법
```bash
python analyze.py <파일경로>          # 사람이 읽기 좋은 리포트
python analyze.py <파일경로> --json   # JSON으로 출력 (다른 도구에 연결용)
```

지원 형식: `.txt` `.md` `.markdown` `.csv` `.json` `.docx`

## 동작 방식 (픽셀 아닌 '구조' 분석)
1. 확장자로 파서를 고릅니다.
2. 공통 통계(줄·단어·문단 수)를 내고,
3. 형식별로 추가 분석:
   - **md**: 헤딩 트리, 코드블록·링크·표 개수
   - **csv**: 행·열 수, 헤더 컬럼명
   - **json**: 최상위 타입과 키 구조(스키마)
   - **docx**: 본문 문단 수, 표 개수 (zip+XML 파싱)
