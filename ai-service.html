KEY IDEA

GitHub Pages는 인터넷에 생긴 ‘내 집’이에요

내가 만든 화면을 공짜로 인터넷에 올려서 ‘집 주소(URL)’를 주는 서비스예요. 누구나 그 주소로 우리 집에 놀러올 수 있죠. 그런데 이 집엔 주방이 없어요. 그래서 진짜 AI 요리는 배민으로 거대한 식당(Grok)에 시켜서 받아옵니다.

GitHub Pages · 내 집

주방 없는 가정집

인터넷에 공짜로 생긴 내 집이에요. 손님(방문자)에게 화면·버튼은 보여줄 수 있지만, 주방이 없어서 직접 요리(AI 연산)는 못 해요. 그래서 배민으로 시킵니다.

배민 주문
Grok (xAI) · 배민 맛집

배민으로 시키는 거대한 식당

주문을 받아 진짜로 요리(AI 연산)하고, 완성된 답을 배달해 줍니다. 단, 출입증(토큰)이 있어야 주문할 수 있어요.

‘배민’ 역할 = API

여기서 배민처럼 주문을 전달해 주는 게 바로 API예요. 정해진 방식으로 주문을 주고받기로 한 ‘약속’이죠.

SECURITY FIRST

출입증(토큰)을 코드에 적으면 안 됩니다

우리 저장소는 누구나 볼 수 있는 공개(public) 저장소예요. 코드에 토큰을 넣으면, 모르는 사람이 그 출입증으로 ‘내 돈을 써서’ 주문할 수 있습니다.

위험한 방법

코드에 토큰을 박아 넣기

HTML/JS 파일에 xai-...를 직접 적고 커밋. → 전 세계에 공개되어 도용·과금 폭탄 위험.

우리가 쓸 방법 (BYOK)

쓰는 사람이 직접 입력

화면의 입력칸에 각자 자기 토큰을 붙여넣어요. 그 키는 본인 브라우저에만 저장(localStorage)되고 저장소엔 절대 안 올라갑니다.

BYOK = Bring Your Own Key (자기 키 가져오기)

개인·학습용으로 가장 간단하고 안전한 방식입니다. 나중에 ‘아무나 써도 되는’ 진짜 서비스로 키우려면 작은 중계소(프록시 서버)를 따로 두면 돼요. 지금은 BYOK로 충분합니다.

STEP 1 · 직접 할 일

Grok 토큰 발급받기

  1. console.x.ai 접속 → 이메일이나 구글 계정으로 로그인/가입
  2. 카드 없이 무료로 시작하기: 왼쪽 Settings → Data Sharing 에서 Share API Inputs for Model Training을 켜면 무료 크레딧이 들어옵니다(보통 24시간 안에). 내 대화가 모델 학습에 쓰이는 대신 무료예요. 잔액은 콘솔에서 확인.
  3. 왼쪽 메뉴 API Keys 클릭
  4. Create API key → 이름은 아무거나 (예: asanax-learning)
  5. xai- 로 시작하는 긴 문자열이 나오면 메모장에 안전하게 복사. 딱 한 번만 보여줍니다.
토큰은 비밀번호예요

채팅창·메신저·저장소 어디에도 붙여넣지 마세요. 3단계에서 만들 화면의 입력칸에만 넣습니다.

# 터미널에 붙여넣기 curl https://api.x.ai/v1/chat/completions \ -H "Authorization: Bearer xai-..." \ -H "Content-Type: application/json" \ -d '{"model":"grok-4","messages":[{"role":"user","content":"안녕!"}]}'

노란 칸(xai-...)에 발급받은 내 토큰을 그대로 붙여넣기 · 답이 오면 통과!

STEP 4 · 로컬 테스트

먼저 내 컴퓨터에서
확인해 봐요

Claude Code가 만든 ai-service.html을 더블클릭해 브라우저로 열면, 내 토큰이 이미 코드에 들어 있어 바로 대화가 됩니다. 내 컴퓨터에서만 돌리는 단계예요.

답이 안 오거나 에러가 나면 Claude Code에게 “이런 에러가 나, 고쳐줘”라고 화면 내용을 그대로 전하면 됩니다. 이 파일은 토큰이 들어 있으니 아직 깃허브에 올리면 안 돼요.

BEFORE DEPLOY · 토큰 숨기기

배포할 땐 토큰을 어떻게 숨길까?

솔직히 말할게요. GitHub Pages처럼 ‘화면 코드’만 있는 사이트에선, 토큰을 변수에 넣어도 누구나 F12로 열어보면 그대로 보입니다. 그래서 ‘코드에 토큰 넣고 배포’는 금지. 진짜로 숨기는 방법은 두 가지예요.

쉬움 · BYOK

배포본엔 토큰을 아예 안 넣기

배포 버전은 입력칸만 두고, 쓰는 사람이 자기 키를 넣게 합니다. 내 토큰은 인터넷에 안 올라가요. 개인·학습용으로 가장 간단합니다.

확실 · 프록시

중계소에 ‘진짜 비밀’로 숨기기

Cloudflare Workers 같은 무료 서버에 내 토큰을 환경변수(진짜 비밀)로 저장합니다. 화면은 그 중계소에 부탁하고, 중계소가 토큰을 붙여 Grok에 요청해요. 토큰이 화면에 안 나오고, 남들도 쓸 수 있죠.

‘변수로 숨기기’가 진짜 통하는 곳

환경변수에 숨기는 건 프록시(서버)에서만 통합니다. 정적 사이트에선 안 돼요. 우리는 다음 칸에서 BYOK로 간단히 갈게요. (로컬 테스트용 키가 든 파일은 .gitignore에 넣어 커밋에서 빼면 됩니다.)
